Load large CSV files, auto-detect encoding & delimiter, plot series (overlay or separate) with axis limits, and export quick statistics. Includes a terminal TUI and a Streamlit web UI.

  • Smart encoding/header detection
  • Plotly charts (overlay / per-series)

SAM — Smart Agriculture Monitor

IoT monitoring stack using ESP32 nodes to collect ambient and soil telemetry, sync data to a Python pipeline, and surface live status inside a Streamlit dashboard.

  • Multi-sensor ingestion with calibrated thresholds
  • Automated alerts for irrigation and climate drift
  • Prototype dashboard with real-time charts and logs

ModelGen — CSV Model Generator

End-to-end CLI/GUI tool that turns tabular datasets into deployable models with automated preprocessing, baseline RandomForest training, and artifact exports for downstream teams.

  • Robust delimiter & encoding detection
  • Configurable preprocessing (imputation, OneHotEncoder)
  • Exports models, predictions, reports, and quick stats
